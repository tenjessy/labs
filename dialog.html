easyDialog.open({
  container : {
    header : '跟随定位',
    content : '弹出窗口可以跟随某元素绝对定位'
  },
  follow : 'demoBtn5',
  followX : -137,
  followY : 24
});

follow：被跟随元素，参数可以是元素的id，也可以是DOM对象。
followX：相对于被跟随元素的X轴偏移。
followY：相对于被跟随元素的X轴偏移。
